### Goals

Girls Who Code holds weekly club meetings through zoom covering the topics of HTML, JavaScript, and CSS. Through these lectures, you can gain the knowledge necessary to create personal websites and interactive pages such as personality quizzes. You can also learn how to use Python to solve logic problems and create Mad Libs, and Pygame to create simple moving characters and backgrounds.
